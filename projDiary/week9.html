♦ What is your final project about?

My Week 8 entry goes into quite a bit of detail, but it's basically a colour generator for charts.

The basic features are:

• being able to generate random colours by clicking a button

• being able to see the generated colours on a chart preview (line chart only, fixed data that cannot be edited on the site)

• being able to click on individual colour swatches to copy the hex codes.

Some optional features that I am looking to add are (in order of priority):

• allowing users to save an image of the chart preview and colour palette (can use canvas.toBlob I think?)

• allow previews of different chart types either through radio buttons or separate pages (radio buttons preferred)

• locking certain colours so that it doesn’t change while the others do

• let people change the data to manipulate the chart

For this project, I am currently using random placeholder values for the chart as it is only needed as a preview. However, I may make a feature that allows for the user to enter their own data if there is time.

A very basic, ugly version of the app has been partially completed if it's needed for reference!
