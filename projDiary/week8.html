♦ Project Title: Charteruse

This title is a play on words from the colour Chartreuse. I thought it was interesting since it's one of the only colours I know with the word chart in it!

This project is going to be a colour generator for charts.

♦ What does a colour generator do?

A colour generator helps people pick colours by generating random colours.

As this project is aimed at colour generation for charts specifically, this means that it would help to have a preview.

♦ What are the "rules" of this generator?

I think there should be sufficient colour contrast between the different colours, which can be ensure through maths. I am also looking to apply colour theory so that the palettes come out pleasing.

However, if I cannot find/figure out the maths to apply, I think I will go with the scaling function that we learnt in the Colour Generator practice because that function generates very nicely spaced colours along a gradient.

♦ What can the user do?

The basic functions of the colour generator are:

• being able to generate random colours by clicking a button

• being able to see the generated colours on a chart preview (line chart only, fixed data that cannot be edited on the site)

• being able to click on individual colour swatches to copy the hex codes.

Some optional features that I am looking to add are (in order of priority):

• allowing users to save an image of the chart preview and colour palette (can use canvas.toBlob I think?)

• allow previews of different chart types either through radio buttons or separate pages (radio buttons preferred)

• locking certain colours so that it doesn’t change while the others do

• let people change the data to manipulate the chart
