For the final project, I am planning to build a simple colour generator for linecharts.

Motivations

Colours are an important communication tool in any visual design. But they can be tricky to work with and colour-picking takes time to perfect, even with a background in design and colour theory. I want to build something that can ease the process of picking colours for data visualisations. On a personal note though, I just really like colours and wanted to find a way to focus on them in the final project.

Objectives

The goal of the app is to make it easier for people to use aesthetic and accessible palettes for their linecharts. Linecharts are one of the most popular graphs, which is why the app focuses on them. To reach this goal, a number of sub-objectives have to be achieved:

• generating colour palettes

• making sure the colours generated contrast well against each other

• feature that allows designer to preview the chart with colour-blind filters so they can make an informed decision

• allowing people to export the palettes/previews, whether through copypasting or downloading an image (even better, both!)

• allow people to "lock" certain colours when generating palettes so that locked colours don't change

• allowing people to input their own data so that the linechart preview can be customised (low-priority & optional because I'm not sure if I have the time to work this out)

Data + Analysis

In this section, I first highlight what I need to track about the user. This is followed by a brief description on what js I can use to get the desired function.

The first, most obvious data needed is to know whether the user clicked on the "generate" button so that the colour palette will change. This can be done with an eventListener.

Depending on the algorithm being used to generate the colours, there may be a need to know how many times the user has generated the palette so that a new set of random values can be generated. This can be done with a function and an if statement.

If I want users to be able to download the image, then I will also need a button that can detect if the user wants this.

If I want users to be able to lock colours, I will also need a button to detect this. In the js, I will then need an "if statement" to determine if each colour is locked or not before generating a new random colour for it.
